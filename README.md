## Informed Systems LLC Website

Static, responsive marketing site for Informed Systems LLC.

### Tech
- HTML5, CSS3, vanilla JavaScript
- No build step; serves as static files

### Structure
- `index.html` (Home)
- `about.html`
- `services.html`
- `contact.html`
- `assets/css/styles.css`
- `assets/js/main.js`

### Local preview
Open `index.html` in your browser. For a simple server:

```bash
python3 -m http.server 8080
# then visit http://localhost:8080
```

### Deployment
- Commit to the GitHub repo `mikedoucettejr/informed-systems-llc`.
- Enable GitHub Pages for the repository (deploy from the `main` branch / root).
