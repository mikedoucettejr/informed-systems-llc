## Informed Systems LLC Website

Personal site for Mike Doucette and Informed Systems LLC — Data Systems Design, Technical Consulting, & Creative Infrastructure.

### Tech
- HTML5, CSS3, vanilla JavaScript
- Google Fonts: IBM Plex Mono + Inter
- No build step; serves as static files

### Structure
- `index.html` (Home)
- `projects.html` (Client Projects & Studio Work)
- `about.html` (About, Services, Resume combined)
- `contact.html` (Email & LinkedIn)
- `assets/css/styles.css`
- `assets/js/main.js`

### Design
- Minimal, technical aesthetic
- Monospace (IBM Plex Mono) + Sans Serif (Inter)
- Clean typography, subtle borders
- Responsive layout

### Local preview
Open `index.html` in your browser. For a simple server:

```bash
python3 -m http.server 8080
# then visit http://localhost:8080
```

### Deployment
- Commit to the GitHub repo `mikedoucettejr/informed-systems-llc`.
- Enable GitHub Pages for the repository (deploy from the `main` branch / root).
- Site will be live at: `https://mikedoucettejr.github.io/informed-systems-llc/`
